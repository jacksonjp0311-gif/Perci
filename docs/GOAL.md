# Perci goal

**Date:** 2026-07-17  
**Software:** v0.6.21+

## Goal (banner)

> **Perci’s goal is not to sound smarter. It is to detect its own failures, open bounded repairs, prove transfer under paraphrase and novel entities, merge only green code, and never silently rewrite truth or weights.**

## Engineering target

A **governed local cognitive OS** (Local AGI-candidate), not AGI, not consciousness, not superintelligence.

```text
fail live
  → lab ticket
  → repair correct layer (operator / tool / reflex / curriculum)
  → transfer suite + hardness + cargo test
  → merge green code (optional agent --merge-if-green)
  → weights only with human --authorize
```

## Non-goals

- Consciousness claims from geometry telemetry  
- Auto-promote `.pwgt` from ledger or tickets  
- Fluency without transfer gates
