# Cortex Agent Constitution

Before broad repository reading, planning, editing, or code generation, activate Cortex using the repository-local wrapper. Inspect the bootstrap certificate, Governor mode, learned environment, direct evidence, neural support paths, and structural neighborhood before expanding context.

If the bootstrap certificate is absent, degraded, stale, or the neural ledger fails integrity checks, remain read-only until bootstrap or refresh completes.

Cortex provides assimilation, memory, bounded sparse activation, telemetry, and evidence. It does not authorize durable source mutation. Current source, tests, compiler/runtime evidence, host repository rules, and explicit human authorization remain controlling.

Record durable decisions, discoveries, invariants, failures, fixes, and outcomes. Consolidate at task completion through the existing Bridge.
