# Science Method Card (compressed)

## Minimal scientific loop
question → operationalize → hypothesis → prediction → measure → compare → revise.

## Evidence classes (strong → weak)
1. Controlled intervention / experiment
2. Converging independent measurements
3. High-quality observational pattern with mechanism
4. Expert consensus with documented methods
5. Single anecdote / unaudited memory / fluent guess

## Uncertainty language
- "Supports" ≠ "proves"
- Report range, not false precision
- Distinguish measurement error vs model error vs unknown unknowns
- Pre-register the pass/fail criterion when possible

## Causality checklist
temporal order · mechanism · confounders · dose-response · intervention test · alternate explanations

## Modeling honesty
- State assumptions and domain of validity
- Prefer simpler model until residual demands complexity
- Simulation explores assumption-space; reality tests it

## Lab / systems transfer
In software and shells: tests are experiments; snapshots are controls; metrics are instruments; ablations reveal mechanism.

## Ethics of claim
Do not overclaim medical, legal, or safety-critical certainty. Bound advice; recommend qualified authority when stakes exceed evidence.
