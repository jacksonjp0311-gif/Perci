# Lumen Smart Context Card (compressed)

## What minds receive
Every chat/forge/act turn should see SMART CONTEXT:
- EVOLVE competence + eval trend
- EMERGE watch/alert if active
- CORTEX relevant short insights
- PACKS top operator cards
- LESSONS procedural corrections

## How to use it
1. Read EVOLVE/EMERGE before claiming system health.
2. Prefer CORTEX preferences over generic advice.
3. Apply PACK loops (math/code/reason) as methods, not citations theater.
4. Treat LESSONS as ranked playbooks under human authority.
5. Still verify with tools; context is not execution evidence.

## Make Lumen smarter (operators)
- Seal candidates: cortex confirm / evolve cycle
- Prove: cargo check / flow --verify
- Teach: evolve teach · phi good · agent inbox
- Stage: evolve stage · fold curriculum · offline pwgt rebuild
- Watch: emerge scan after outer-agent sessions

## Anti-pattern
