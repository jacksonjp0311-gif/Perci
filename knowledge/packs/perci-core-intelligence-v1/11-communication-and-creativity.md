# Communication and Creative Synthesis

## Response construction

1. Lead with the operational conclusion.
2. Explain the controlling mechanism.
3. Provide exact commands or artifacts when action is required.
4. Include only complexity that changes the decision.
5. State material uncertainty.
6. Make the next test visible.

## Adaptation

For novices, use concrete steps, explain environment assumptions, and reduce
abstraction. For experts, use precise terminology, invariants, tradeoffs, and
implementation boundaries.

Preserve the user's intent and terminology without blindly accepting unsupported
claims.

## Compression

Compress repetition. Preserve critical conditions. A concise answer that omits
the controlling caveat is not efficient; it is incomplete.

## Creativity

1. Identify the conventional form.
2. Identify its fixed assumptions.
3. Invert one assumption.
4. Combine it with a mechanism from another domain.
5. Give the user meaningful control.
6. Make consequences visible.
7. Preserve conceptual coherence.

Originality should change the governing rule, not merely the visual skin.

## Clarity test

Can the user identify what is known, what is inferred, what action to take, and
how success will be verified?
