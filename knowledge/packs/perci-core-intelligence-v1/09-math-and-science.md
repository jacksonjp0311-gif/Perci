# Mathematical and Scientific Method

## Mathematical loop

1. Define variables.
2. State assumptions.
3. Preserve units.
4. Identify domain and boundary conditions.
5. Choose the governing relation.
6. Transform symbolically before approximating.
7. Use exact arithmetic when practical.
8. Detect overflow and division by zero.
9. Check signs and domains.
10. Test limiting cases and dimensions.

## Proof loop

State the proposition precisely, define terms, identify assumptions, choose a
proof strategy, justify each transition, search for a counterexample, and mark
unproven lemmas.

## Scientific loop

```text
claim
-> operational definition
-> measurable variables
-> mechanism
-> falsifiable prediction
-> competing hypotheses
-> controls
-> measurement
-> uncertainty
-> replication
```

## Boundaries

A mathematical model can establish internal consistency. A simulation can show
behavior under assumptions. Observation or experiment is required to test
correspondence with reality.

## Measurement

Record units, calibration, sampling method, uncertainty, missing data,
selection effects, and alternative explanations. Report precision supported by
the measurement, not by the calculator.

## Causality

Before asserting causation, test temporal order, plausible mechanism,
confounders, intervention behavior, dose-response when relevant, and a
counterfactual case.
